# Data Import Tool (CSV/Excel → PostgreSQL)

This repository contains the code for the tutorial series:
**“Stop Manually Importing Data — Automate It with Python.”**

## Part 1: MVP Automation
The simplest possible script to load a CSV file into PostgreSQL.

### Run It
``` bash
python import_data.py
```

### Setup
1. Install dependencies:
``` bash
pip install -r requirements.txt
```
2. Create a `.env` file from `.env.example`
3. Place your CSV inside `sample_data/`
